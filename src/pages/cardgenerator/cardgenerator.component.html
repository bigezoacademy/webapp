HOW TO USE

This software enables you to customize invitation cards. It types recipients' names on a card designed by you. Just leave space for the names and create a CSV (Comma Separated Value) files using MS Excel or a text editor like notepad. For Notepad or Notes Apps, Just make sure the names are separated using commas, then save it with a file extension .csv after the file name, e.g. NamesList.csv -Lastly, when pasting the file path, remove the "", let the path just be like C:\Users\dell\Documents\CARD\page1.jpg
